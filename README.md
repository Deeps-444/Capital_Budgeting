# Explainable Decision Support System for Capital Budgeting

## Project Overview

This project implements an explainable decision support system for capital budgeting using financial models and machine learning.

The system evaluates investment projects based on:

- Initial investment
- Projected cashflows
- Risk assessment
- Discount rate

Machine learning is used to **learn and explain risk patterns**, while final investment decisions are derived using transparent financial rules.

---

## System Architecture

1. Risk Score Prediction (Linear Regression – Explainable)
2. Financial Calculations (NPV, IRR, Payback Period)
3. Rule-Based Decision (Accept / Reject)
4. (Planned) Java Backend Integration
